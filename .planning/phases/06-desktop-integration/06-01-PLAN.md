Update `systemdUnitTemplate` to add an `ExecStartPost` line that sets ACL on the companion's session bus socket, granting the desktop user read/write access. The template needs `DesktopUser` as a new template variable:

```

```

Add it after `ExecStart=`. This runs every time the companion daemon starts, ensuring the ACL is re-applied even after reboot (tmpfs at /run/user/ is recreated).

Add the `DesktopUser` field to the template vars struct in `writeSystemdUnit()`.
